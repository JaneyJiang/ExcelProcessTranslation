首先分两个功能1.Similarity（根据相似度对输入的数据进行划分） 2.合并表（根据给定的一定格式的几张表进行合并来得到所有翻译结果）
相似表功能中:
1.Load 用语集表，以excel方式载入用语集，一一对应的方式
2.Load 带翻译表，以excel方式载入我们需要翻译的词条和对应的uniqueId
3.outoput Tables这张表会输出两张表，一个是合并时需要用到的索引表（这个不能改动），一个是包含需要人工翻译的简表

合并表功能中：
1.Load 索引表
2.Load待翻译表（总表）
3.Load简表（即在相似表功能中输出的简表（添加过人工翻译））
4.Output：输出根据一定程度人工翻译可以获得的总表的翻译效果。
